Здравствуйте!
Здесь вы можете посмотреть все доступные вам опции и фильтры которые вы можете настроить
/start - инициализирует приложение и является входной точкой в приложение
/help - команда является документацией пользователю по всем доступным опциям
/run - запускает приложение для поиска квартир/домов
/cfg - останавливает команду /run и дает возможность поменять значение фильтров
/city {city} - команда отвечает за выбор города откуда будеть происходить выборка квартир.
Но для этой команды неоюходимо правильно указать город.
Пример: /city Алматы команда принимает только один город начинается с большой буквы

Опции для покупки квартир
/room {room's} - в этой команде вы указываете квартиры с числом комнат.
Указывайте комнаты через разделитель (,).
Пример 1,3,4.
Для поиска 5 и более комнатных квартир введите 5+. Пример 1,5+
/type {house's type} - в этой команде вы указываете тип дома.
Доступные варианты [кирпичный, панельный, монолитный]. Указывать через разделитель (,).
Пример /type кирпичный,монолитный
/builf {year} - команда принимает одно значение стартовый год квартир/домов которые вас интересуют
Пример /builtf 2015.
/builtt {year} - команда принимает одно значение. До какого года квартиры/дома которые вас интересуют
Пример /builtt 2015.
/pricef {price} - команда принимает стартовую цену квартир/домов которые вас интересует.
Пример /price 10 000 000
/pricet {price} - команда которая принимает конечную цену квартир/домов которые вас интересуют
Пример /price 10 000 000
/floorf {floor} - команда принимает число. Квартиры с минимальным этажем
/floort {floor} - команда принимает число. Квартиры с максимальным этажем
/checknf - введите эту команду для поиска квартир от 2 этажа
/checknl - введите эту команду для поиска квартир до предпоследнего этажа
/checkfo - введи эту команду для поиска квартир только от хозяев
/checknb - введи эту команду для поиска квартир только от новостроек
/checkre - введи эту команду для поиска квартир только от крыша агентов
/areaf {area} - команда принимает целое цисло и является минимальным значением площади квартиры
/areat {area} - команда принимает целое число и является максимальным значение площади квартиры
/kitchenf {area} - команда принимает целое число и является минимальным значение площади кухни
/kitchent {area} - команда принимает целое число и является максимальным значение площади кухни
