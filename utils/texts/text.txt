Приветсвуем вас в документации сервиса
Ознакомьтесь с документацией очень внимательно
Здесь вы можете посмотреть все доступные вам опции и фильтры которые вы можете настроить
/start - инициализирует приложение и является входной точкой в приложение
Команда является обязательной!!!
/help - команда является документацией пользователю по всем доступным опциям
/run - запускает приложение для поиска квартир/домов
/stop - останавливает команду /run и дает возможность поменять значение фильтров и затем снова начать поиск по команде /run
/city - команда отвечает за выбор города откуда будеть происходить выборка квартир.
Но для этой команды неоюходимо правильно указать город.
Пример: /city Алматы команда принимает только один город начинается с большой буквы
Команда также является обязательной!!!
/all - команда для просмотра всей конфигурации ваших фильтровr5

Опции для покупки квартир
/rooms {room's} - в этой команде вы указываете квартиры с числом комнат.
Указывайте комнаты через разделитель (,).
Пример 1,3,4.
Для поиска 5 и более комнатных квартир введите 5+. Пример 1,5+
/type {house's type} - в этой команде вы указываете тип дома.
Доступные варианты [кирпичный, панельный, монолитный]. Указывать через разделитель (,).
Пример /type кирпичный,монолитный
/builtFrom {year} - команда принимает одно значение стартовый год квартир/домов которые вас интересуют
Пример /built-from 2015.
/builtTo {year} - команда принимает одно значение. До какого года квартиры/дома которые вас интересуют
Пример /built-to 2015.
/priceFrom {price} - команда принимает стартовую цену квартир/домов которые вас интересует.
Пример /price-from 10 000 000
/priceTo {price} - команда которая принимает конечную цену квартир/домов которые вас интересуют
Пример /price-to 10 000 000
/floorFrom {floor} - команда принимает число. Квартиры с минимальным этажем
/floorTo {floor} - команда принимает число. Квартиры с максимальным этажем
/notFirst - введите эту команду для поиска квартир от 2 этажа
/notLast - введите эту команду для поиска квартир до предпоследнего этажа
/fromOwner - введи эту команду для поиска квартир только от хозяев
/newBuilding - введи эту команду для поиска квартир только от новостроек
/realEstate - введи эту команду для поиска квартир только от крыша агентов
/floorHouseFrom {floor} - команда принимает число минимальная этажность дома
/floorHouseTo {floor} - команда принимает число максимальная этажность дома
/areaFrom {area} - команда принимает целое цисло и является минимальным значением площади квартиры
/areaTo {area} - команда принимает целое число и является максимальным значение площади квартиры
/kitFrom {area} - команда принимает целое число и является минимальным значение площади кухни
/kitTo {area} - команда принимает целое число и является максимальным значение площади кухни
